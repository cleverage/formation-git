Préparer son poste pour une formation Git

Afin de préparer au mieux la formation Git de Clever Institut, voici quelques pré-requis nécessaires pour disposer d'une version récente de Git sur votre poste.

Sur Linux

La liste des paquets à installer (avec le gestionnaire de votre choix) est la suivante :
  • git-core;
  • git-docs, qui fournit la documentation;
  • git-completion, qui permet d'utiliser la complétion automatique des commandes.
  • gitg, qui permet d'utiliser une interface graphique pour visualiser l'historique.

Sur certaines versions de Linux, le packet git-core a été taggé comme déprécié. Il faut installer le packet git qui fournit automatiquement git-core et git-completion

Même si les cycles de sortie et de maintenance peuvent varier suivant la version de Linux que vous utilisez, les versions de Git qui leur sont attachées sont bien souvent à la traine... La meilleure façon de disposer de la dernière version de Git sur votre machine reste encore de compiler directement les sources.

Sur Mac OSX

Pour installer Git sous Mac OSX, deux solutions s'offrent à vous :
  • Comme avec Linux, vous pouvez utiliser un gestionnaire de paquets, Homebrew. Il vous suffit alors d'utiliser la commande brew install git...;
  • Vous pouvez également installer Git en utilisant Git OSX Installer. Régulièrement mis à jour, l'installeur fournit généralement une version très proche de la version publique de Git.

Sur Windows

Le mieux pour installer Git sous Windows est de télécharger MsysGit, et de suivre les étapes pas à pas.

Pour vérifier que l'installation a fonctionné correctement, exécutez la commande git --version. Si aucun message d'erreur n'apparait, c'est que votre version de Git est correctement installée !

Si vous êtes derrière un proxy

Si vous êtes derrière un proxy, vérifier que vous avez accés aux domaines github.com et raw.githubusercontent.com
