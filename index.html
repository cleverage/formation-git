Préparer son poste pour une formation Git

Afin de préparer au mieux la formation Git de Clever Institut, voici quelques pré-requis nécessaires pour disposer d'une version récente de Git sur votre poste.

Sur Linux

La liste des paquets à installer (avec le gestionnaire de votre choix) est la suivante :
  • git-core
  • git-docs
  • git-completion
  • git

Même si les cycles de sortie et de maintenance peuvent varier suivant la version de Linux que vous utilisez, les versions de Git qui leur sont attachées sont bien souvent à la traine... La meilleure façon de disposer de la dernière version de Git sur votre machine reste encore de compiler directement les sources.
